# 🧠 AI Study Quiz & Flashcard Generator

An interactive, AI-powered web application built with **Streamlit** and **Google Gemini API** that instantly transforms lecture notes, textbook chapters, or articles into custom practice quizzes and study flashcards.

## 🚀 Features
* **Instant Quiz Generation:** Automatically creates multiple-choice questions with 4 options and correct answers indicated from any study text.
* **Flashcard Creation:** Generates key concepts and definitions for effective studying.
* **Customizable Length:** Use the sidebar slider to choose between 3 to 10 questions.
* **Secure API Integration:** Integrates dynamically with Google Gemini API keys via a secure user interface.

---

## 🛠️ Tech Stack
* **Python**
* **Streamlit** (Web Application Framework)
* **Google Generative AI SDK** (`gemini-2.5-flash`)
