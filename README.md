**AI Study Quiz & Flashcard Generator**

## a. App Name, What It Does, and Real Problem Solved
* **App Name:** AI Study Quiz & Flashcard Generator
* **What it does:** An interactive, AI-powered web application that instantly transforms lengthy lecture notes, textbook chapters, or articles into structured practice quizzes and concise study flashcards.
* **Problem it solved & Target Audience:** Students, educators, and lifelong learners often spend hours manually creating practice questions and flashcards from dense study materials. This app automates active recall preparation, saving hours of manual study design and enhancing learning efficiency.

## b. Live Deployed URL
* **Live App:** [Click here to view the live Streamlit app](https://ai-quiz-generator-dwojjufcn73hua7m6eetvz.streamlit.app)

## c. Features List
* **Instant Quiz Generation:** Automatically generates multiple-choice questions with 4 distinct options and clear answers from any provided text.
* **Flashcard Creation:** Extracts critical concepts and definitions to facilitate memorization.
* **Customizable Length:** Features a sidebar slider allowing users to adjust the quiz length from 3 to 10 questions.
* **Secure Cloud Architecture:** Implements secure configuration management to protect API credentials.

## d. AI Feature & System Instructions
* **AI Feature:** Utilizes generative language understanding to parse, summarize, and format raw educational text into pedagogical quiz questions and definitions.
* **Prompt Structure / System Instructions:** 
  > "Based on the following text, generate a practice quiz with multiple-choice questions (with 4 options each and the correct answer indicated) and flashcards with key concepts and definitions."

## e. Tools, Services, and AI Models Used
* **Programming Language:** Python
* **Web Framework:** Streamlit
* **AI Model:** Google Gemini (`gemini-1.5-flash`) via Google Generative AI SDK
* **Hosting & Deployment:** Streamlit Community Cloud
* **Version Control:** GitHub
